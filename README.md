# 🌸 Lyra's story arc (Moana movie inspired)
Each break, Lyra learns something new and her XP grows. After every 4 sessions she "levels up" - new dialogue unlocks. She starts as a Novice Warrior and grows into a Legend. The skill she learns is woven into her dialogue — "Guardian, I just learned about recursion. Even quests loop back sometimes…"

# 🌸⚔️ Lyra's Quest is BUILT! Here's how to load it right now:
## 🚀 Install in Chrome (2 minutes)

1. Download the ZIP above and unzip it to a folder
2. Open Chrome → go to `chrome://extensions`
3. Toggle "Developer mode" ON (top right)
4. Click "Load unpacked" → select the `lyra-quest` folder
5. Click Lyra's icon in your toolbar — she's alive! 🌿
   
## 🎮 What you just built — full feature list:
Timer engine — 25 min work / 5 min break Pomodoro, survives tab switches, runs in the background
Lyra's fullscreen takeover — she walks onto your screen, collapses dramatically, then jumps up restored when break ends. Cannot be dismissed until the last 5 seconds
50 skills across coding, philosophy, fitness & wildcards — randomly picked every break, never repeating recently seen ones
XP + leveling system — 6 levels from Novice Wanderer → Legend of the Realm, with a level-up banner that fires when you rank up
Live popup with timer ring, XP bar, session count, skills learned count, and last skill
Lyra's own dialogue — every skill card has her in-character quote woven into the lesson, Moana-style 🌊
